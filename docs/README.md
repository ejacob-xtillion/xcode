## Documentation Guidelines

When updating documentation:

1. **Keep CLAUDE.md in sync** - This is critical for AI assistants
2. **Update README.md** - Reflect major architectural or feature changes
3. **Add to CHANGELOG.md** - Document all notable changes
4. **Update CONTRIBUTING.md** - Document new development workflows
5. **Create reports in docs/** - Keep technical reports organized here
